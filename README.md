### 文件说明

* Power001 包含CUBEMX配置文件、主程序。（MATLAB生成代码在MATLAB.rar中，需把这两个文件夹放在同一目录下）
* Shell 包含外壳建模文件、串口屏工程图。
* Screen 包含串口屏程序、字库文件、用到的logo。
* MATLAB 包含MATLAB参数文件、Simulink的pid模型文件。
* 测试方案与测试结果.pdf 调试时的测试图片（与赛题评分时测试内容不同）。
